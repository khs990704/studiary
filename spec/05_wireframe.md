## 2. 사용자 흐름

```mermaid
flowchart TD
    Start([서비스 접속]) --> AuthCheck{로그인 상태?}
    AuthCheck -->|아니오| Login[로그인 페이지]
    AuthCheck -->|예| Main[메인 페이지]
    Login -->|회원가입 링크| Register[회원가입 페이지]
    Register -->|가입 성공| Login
    Login -->|로그인 성공| Main

    Main -->|오늘 카드 '공부하러가기' 클릭| StudyInit[공부 기록 - 초기 상태]
    Main -->|과거 카드 클릭| StudyReview[공부 기록 - 리뷰 상태 읽기전용]
    Main -->|히트맵 날짜 클릭| ScrollCard[해당 카드로 스크롤]

    StudyInit -->|'+' 클릭| TimerSetup[타이머 설정]
    TimerSetup -->|시작 클릭| StudyRunning[공부 기록 - 진행 중]
    StudyRunning -->|타이머 완료| SessionDone[세션 완료 표시]
    SessionDone -->|'+' 클릭| TimerSetup
    StudyRunning -->|'오늘 공부 끝내기'| FinishFlow[종료 처리]
    SessionDone -->|'오늘 공부 끝내기'| FinishFlow

    FinishFlow --> AICall{AI 호출}
    AICall -->|성공| ReviewWithAI[리뷰 + AI 요약/피드백]
    AICall -->|실패| ReviewNoAI[리뷰 + 재생성 버튼]

    ReviewWithAI -->|'<' 뒤로가기| Main
    ReviewNoAI -->|'<' 뒤로가기| Main
    ReviewNoAI -->|재생성 클릭| AICall

    StudyReview -->|'<' 뒤로가기| Main
```

## 4. 세션 상태 다이어그램

```mermaid
stateDiagram-v2
    [*] --> TimerSetup: '+' 버튼 클릭
    TimerSetup --> Running: 시작 클릭 (세션 생성 API)
    Running --> Paused: 일시정지 클릭
    Paused --> Running: 재생 클릭
    Running --> Completed: 타이머 시간 완료
    Completed --> [*]: 세션 완료

    note right of Running
        집중도/방해요소
        수정 가능 (study만)
    end note

    note right of Completed
        재시작 불가
        수정은 당일+미종료까지
    end note
```

## 5. 공부 기록 페이지 상태 다이어그램

```mermaid
stateDiagram-v2
    [*] --> Initial: 세션 0개
    Initial --> HasSessions: '+' → 세션 생성
    HasSessions --> HasSessions: '+' → 세션 추가
    HasSessions --> HasSessions: 세션 삭제
    HasSessions --> Finished: '오늘 공부 끝내기'
    Initial --> NoSessionAlert: '오늘 공부 끝내기' (세션 0개)
    NoSessionAlert --> Initial: 알림 닫기

    Finished --> ReviewWithAI: AI 성공
    Finished --> ReviewNoAI: AI 실패
    ReviewNoAI --> ReviewWithAI: 재생성 성공

    note right of Finished
        세션 수정/삭제 비활성화
        is_finished = true
    end note
```

## 6. 히트맵 상세

### 6.1 색상 매핑

| 집중도 평균 올림값 | 색상 | CSS 예시 |
|------------------|------|---------|
| 기록 없음 / 0 | 갈색 | `#d2b48c` |
| 1 | 가장 옅은 초록 | `#c6e48b` |
| 2 | 옅은 초록 | `#7bc96f` |
| 3 | 중간 초록 | `#449945` |
| 4 | 진한 초록 | `#2d6a30` |
| 5 | 가장 진한 초록 | `#196127` |

### 6.2 히트맵 인터랙션

- 월 선택기: 년도/월 변경 시 해당 월 데이터 로드
- 날짜 셀 클릭: 해당 날짜의 Recorded Study Card로 부드러운 스크롤
- 각 셀은 해당 월의 날짜 그리드 (7열 x N행, 일~토)
