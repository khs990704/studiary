# PRD — Studiary: 학습 세션 생산성 기록 및 AI 요약/피드백 서비스

> 버전: 0.1 (spec 초안)
> 최종 업데이트: 2026-04-11

---

## 1. 제품 개요

### 1.1 한 줄 설명
하루 공부 세션을 기록하고, 집중도/방해요소 데이터를 바탕으로 AI가 요약과 피드백을 생성하는 학습 세션 생산성 기록 서비스.

### 1.2 해결하려는 문제
공부 시간 기록 자체에 그치지 않고, 기록된 집중도/방해요소 데이터를 바탕으로 사용자가 자신의 학습 상태를 복기하고 개선할 수 있도록 돕는다.

### 1.3 타깃 사용자
- 공부를 기록하고 개선하고 싶은 학생, 수험생, 직장인
- 자기주도 학습자로 학습 패턴을 데이터 기반으로 분석하고 싶은 사용자

### 1.4 핵심 차별점
- 단순 타이머가 아닌 세션별 집중도/방해요소 기록
- AI 기반 자동 요약 및 피드백 (MVP 핵심 기능)
- 히트맵으로 월별 학습 패턴 시각화

## 2. 기능 요구사항

### P0 (MVP 필수)

| # | 기능 | 설명 | 수락 기준 |
|---|------|------|----------|
| FR-1 | 회원가입/로그인 | 이메일+비밀번호 기반 계정 생성 및 JWT 인증 | 회원가입 후 로그인 시 JWT 토큰 발급, 다중 기기 접속 가능 |
| FR-2 | 메인 기록 화면 | 월별 히트맵 + 날짜별 공부 기록 카드 목록 | 히트맵 색상이 집중도 평균 올림값(1~5) 기반, 카드에 총 시간/AI 요약/피드백 표시 |
| FR-3 | 공부 세션 생성 | 타이머 시간 설정 후 공부/휴식 세션 교대 생성 | 직전 세션이 공부면 휴식, 휴식이면 공부 세션 자동 결정 |
| FR-4 | 세션 진행 기록 | 타이머 진행 중 집중도(1~5) 및 방해요소(100자) 기록 | 공부 세션에만 집중도/방해요소 입력 가능, 당일까지 수정 가능 |
| FR-5 | 세션 수정/삭제 | 당일까지 세션 편집, 삭제 확인 알림 | 삭제 시 확인 다이얼로그, 공부 종료 후 수정 불가 |
| FR-6 | 공부 종료 리뷰 | 세션 목록 + 집중도 변화 그래프 + AI 요약/피드백 | "오늘 공부 끝내기" 클릭 시 리뷰 화면 전환, 세션 편집 비활성화 |
| FR-7 | AI 요약 생성 | OpenRouter 무료 모델로 하루 공부 요약 | 총 공부시간/휴식시간/평균 집중도/방해요소 입력, 1~2문장 요약 생성 |
| FR-8 | AI 피드백 생성 | AI 요약 기반 피드백 생성 | 요약 텍스트 입력, 방해요소 개선/학습 패턴 제안 1~2문장 생성 |
| FR-9 | AI 재생성 | AI 호출 실패 시 재생성 버튼 활성화 | 기록은 저장, AI 결과 없으면 재생성 버튼 표시, 성공 시 비활성화 |
| FR-10 | 과거 기록 조회 | 메인에서 과거 날짜 카드 클릭 시 해당 날짜 기록 로드 | 읽기 전용, 세션 목록/집중도 그래프/AI 결과 표시 |

## 4. 사용자 여정

### 4.1 신규 사용자 첫 공부 기록

1. 회원가입 (이메일/비밀번호)
2. 로그인 후 메인 화면 진입 (빈 히트맵, 오늘 카드에 "공부 하러 가기" 버튼)
3. "공부 하러 가기" 클릭 → 공부 시작 전 초기 화면
4. (+) 버튼 클릭 → 타이머 설정 (예: 50분)
5. "시작" 클릭 → 공부 세션 생성, 타이머 시작
6. 공부 중 집중도 3 선택, 방해요소 "핸드폰 알림" 입력
7. 타이머 완료 → (+) 클릭 → 휴식 세션 10분 시작
8. 휴식 완료 → 추가 공부 세션 반복 또는 "오늘 공부 끝내기" 클릭
9. AI 요약/피드백 생성 → 리뷰 화면에서 확인
10. 메인으로 돌아가면 히트맵에 오늘 기록 반영

### 4.2 기존 사용자 과거 기록 조회

1. 로그인 후 메인 화면 진입
2. 히트맵에서 과거 날짜 클릭 또는 카드 클릭
3. 해당 날짜의 세션 목록, 집중도 그래프, AI 요약/피드백 확인 (읽기 전용)

### 4.3 AI 호출 실패 후 재생성

1. "오늘 공부 끝내기" 클릭 → AI 호출 실패
2. 세션 기록은 저장됨, AI 요약/피드백 영역 비어있음
3. 메인에서 해당 카드 클릭 → 공부 기록 페이지에 "AI 재생성" 버튼 활성화
4. "AI 재생성" 클릭 → AI 요약/피드백 생성 성공 → 버튼 비활성화
